INSTALL
    You should install SDL develop library first, and the project could be opened by visual studio 2010. The sdl libs can be found in ./lib directory. Header files in lib/include/ needs copy to <visual studio>/include, and files in lib/lib needs copy to <visual studio>/lib. The project has been configured to use those libs.
